# Football Data Analysis ⚽🌌

Exploratory data analysis of a fictional football league dataset.

## Overview
Analysis of match data including goals, players and teams, 
covering scoring patterns, age distributions, and 
team performance across tournaments.

## Tools
- Python (pandas, numpy, plotly)

## Techniques
- Data loading and exploration
- Data merging and aggregation
- Exploratory data analysis (EDA)
- Data visualization

## Files
- `football_analysis.ipynb` — full analysis notebook

## Data
Dataset provided by the course instructor. Not publicly available.

## Context
Academic project developed in a postgraduate program 
in Management, Analytics & AI.

# Análisis de Datos de Fútbol ⚽🌌

Análisis exploratorio de un dataset ficticio de una liga de fútbol.

## Descripción
Análisis de datos de partidos incluyendo goles, jugadores y equipos,
cubriendo patrones de gol, distribución de edades y 
rendimiento de equipos a lo largo de los torneos.

## Herramientas
- Python (pandas, numpy, plotly)

## Técnicas aplicadas
- Carga y exploración de datos
- Merge y agregación de datos
- Análisis exploratorio de datos (EDA)
- Visualización de datos

## Archivos
- `football_analysis.ipynb` — notebook con el análisis completo

## Datos
Dataset provisto por el docente de la materia. No disponible públicamente.

## Contexto
Proyecto académico desarrollado en el marco de un posgrado
en Management, Analytics & IA.
